# 메이플스토리 유저 분석

넥슨 오픈API로 수집한 실제 유저 데이터를 바탕으로 경험치 변동 패턴과 유저 행동을 분석합니다.

- [대시보드 바로가기](https://maple-exp-analysis.streamlit.app/)
- [분석 리포트](https://www.notion.so/miniminimin/32afbcdaed288075a929eb7f533361b0?source=copy_link)

## I. 경험치 분석

쇼케이스·이벤트·요일 등 외부 요인이 고레벨 유저(Lv.285~299)의 사냥 동기에 미치는 영향을 검증합니다.

### 1. 쇼케이스 영향 분석

- 쇼케이스 기준일(2025.12.13) ±35일 완전 대칭 구간으로 Pre/Post 비교
- 레벨 구간별(Lv.285~289 / 290~294 / 295~299) 일평균 경험치 변화율 제공
- 대응표본 t-검정으로 구간별 유의성 검증

### 2. 주간 패턴 분석

- 수요일 시작 7일 단위 요일별 평균 경험치 분포 제공
- 주차 × 요일 히트맵으로 시기별 사냥 패턴 시각화
- 메요일(목) / 썬데이(일) 강조 표시

### 3. 썬데이 이벤트 심층 분석

- 썬데이 이벤트를 경타포스 / 사냥 / 사냥 외 3분류로 구분
- 유저별 Pre 평균 대비 상대 경험치 비율로 레벨 구간 간 기저 차이 제거
- One-way ANOVA + Tukey HSD 사후 검정으로 이벤트 유형 간 유의성 검증

### 4. 이벤트 기간 영향 분석

- `eventlog.txt`에 등록된 이벤트에 대해 영향 집계
- 구간별 대응표본 t-검정 결과 제공
- 쇼케이스 이후 일별 추이에 이벤트 기간 하이라이트

## II. 유저 클러스터링

활동 패턴·성장 효율·캐릭터 연령 등 행동 지표를 기반으로 유저를 군집화하고, 이탈 패턴과 스펙 분포를 분석합니다.

### 1. 유저 이탈 분석

- 30일 연속 비활성을 이탈로 정의한 Kaplan-Meier 생존 분석
- 레벨 구간별·클러스터별 이탈 곡선 및 중앙 이탈 시점 비교

### 2. 클러스터 분석

- 활동 일수 비율 / 경험치 퍼센타일 / 유니온 레벨 / 캐릭터 연령 4개 피처로 K-Means 클러스터링 (k=4, silhouette=0.319)
- 엘보우 + 실루엣 기준 최적 k 자동 선택
- 세그먼트 × 클러스터 교차 분포, 쇼케이스 전·후 경험치 변화 비교

### 3. 스펙 분포 분석

- 레벨 구간·월드 그룹별 능력치 바이올린 플롯
- tier × world_group 전투력 중앙값 히트맵
